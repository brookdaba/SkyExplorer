# SkyExplorer
this is a simple video game made using JAVA for ECEC 223(Computer Science II) final project. 
the goal of the game is to fly the plane with out touching any of the dark clouds coming down with from random x positions
Inorder to start the game 
1 - create a java project named Sky_explorer in a java IDE (eclips or NetBeans)
2 - import all the files to the source file of the project
3 - set Sky_explorer.java as the main class 
4 - run the program
